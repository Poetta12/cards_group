21 Months CDA Training Program

Campus Numérique in the Alps - Annecy

Algorithmique

L'algorithmique est le cœur de tout développement informatique. Ce module couvre en profondeur les structures de contrôle comme les boucles et les conditions, ainsi que les structures de données comme les tableaux, les listes chaînées, et les arbres. Vous apprendrez à analyser des problèmes complexes, à créer des algorithmes optimisés pour les résoudre, et à les implémenter dans des langages de programmation. Ces compétences sont essentielles pour optimiser les performances de vos programmes, améliorer la rapidité des traitements, et résoudre des problèmes de grande envergure, comme la gestion de grands volumes de données ou le calcul de chemins optimaux.

HTML/CSS

Ce module est une introduction complète au développement front-end, essentiel pour tout développeur web. Vous apprendrez à structurer des pages web avec HTML, en utilisant des balises sémantiques pour rendre le contenu accessible et optimisé pour les moteurs de recherche. Le CSS sera utilisé pour styliser vos pages, créer des mises en page flexibles et responsives, et ajouter des animations et transitions fluides. Ce module explore aussi les frameworks modernes comme Flexbox et Grid Layout, ainsi que les préprocesseurs CSS comme SASS, qui permettent de rendre vos feuilles de style plus dynamiques et organisées. À la fin de ce module, vous serez capable de créer des interfaces web modernes, optimisées pour tous types d'appareils.

PHP/POO

PHP, l'un des langages de développement web les plus populaires, est au centre de ce module. Vous allez plonger dans la programmation orientée objet (POO) avec PHP, en apprenant à organiser votre code en classes, objets, et méthodes, ce qui permet une plus grande modularité et maintenabilité de vos applications. Vous explorerez des concepts avancés comme l'héritage, le polymorphisme, et l'encapsulation, qui sont cruciaux pour structurer des applications complexes. Ce module vous amène également à comprendre comment PHP interagit avec les bases de données via PDO, et comment créer des API RESTful robustes. À la fin de ce module, vous serez capable de développer des applications web évolutives en suivant les meilleures pratiques de développement orienté objet.

Gestion de Projet

La gestion de projet est un aspect crucial du développement logiciel, et ce module vous équipe des outils et techniques nécessaires pour planifier, organiser, et gérer efficacement un projet du début à la fin. Vous découvrirez les méthodologies traditionnelles comme le cycle en V et les méthodes agiles comme Scrum et Kanban, qui permettent de gérer des projets de manière itérative et collaborative. Vous apprendrez à définir un cahier des charges, à coordonner les équipes, à gérer les risques et à veiller à ce que les livrables respectent les attentes du client. Des outils de gestion comme Jira et Trello seront également présentés pour vous aider à suivre l'avancement des tâches et à prioriser les objectifs. Ce module vous permet d'intégrer une équipe de développement tout en assurant une communication fluide entre les parties prenantes.

Java Spring Boot

Spring Boot est un framework Java extrêmement puissant qui simplifie la création d'applications backend et de microservices. Dans ce module, vous apprendrez à développer des API RESTful avec Spring Boot, à intégrer des bases de données relationnelles comme MySQL via JPA/Hibernate, et à sécuriser vos applications avec Spring Security. Vous explorerez également des concepts avancés comme la gestion des transactions, les architectures microservices, et l'utilisation de Docker pour le déploiement d'applications containerisées. Ce module vous prépare à devenir un développeur backend complet, capable de créer des services robustes et scalables pour des applications distribuées.

CI/CD

Le CI/CD (Continuous Integration / Continuous Deployment) est une méthodologie qui vise à automatiser l'intégration des changements de code et le déploiement en production de manière continue. Ce module vous apprendra à configurer des pipelines CI/CD à l'aide d'outils comme Jenkins, GitLab CI, ou CircleCI, pour automatiser les tests, la construction, et le déploiement de vos applications. Vous apprendrez à mettre en place des tests unitaires, des tests d'intégration, et des tests de bout en bout pour garantir que chaque nouvelle fonctionnalité ou correction de bug fonctionne correctement avant d'être mise en production. En maîtrisant le CI/CD, vous serez capable de réduire le délai de mise en production et de garantir une qualité de code élevée tout au long du cycle de vie du projet.
